# CLI Calculator

A simple command-line calculator built with Node.js that supports basic operations using dynamic command routing.

---

## 🚀 Features

- Add, subtraction, multiply, and divide numbers
- Command-based input system
- Dynamic command map (no long if/else chains)
- Basic error handling for invalid commands

---

Clone the repository:

git clone https://github.com/your-username/cli-calculator.git
